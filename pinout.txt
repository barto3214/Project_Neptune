GND → GND

Czujnik pH:
VCC → 3.3V 
GND/G → GND
Do → A0 na ADS1115


Czujnik przewodności (EC/TDS)(czarna sonda):

VCC/V+ → 5V Raspberry Pi
GND/G → GND
Wyjście analogowe (A/S/Signal) → A1 na ADS1115

Czujnik TDS → ADS1115(biała sonda):

VCC (czerwony) → 5V Raspberry Pi
GND (czarny) → GND
A/SIG (niebieski) → A2 na ADS1115



Podłączenie DS18B20(temp)
Czujnik ma 3 przewody:

Czerwony (VCC) → 3.3V Raspberry Pi 
Czarny (GND) → GND 
Zielony (DATA) → GPIO4 (pin 7)

Silnik krokowy 28BYJ-48 + sterownik ULN2003:
IN1 → GPIO 17 (pin 11)
IN2 → GPIO 27 (pin 13)
IN3 → GPIO 22 (pin 15)
IN4 → GPIO 23 (pin 16)
VCC → 5V Raspberry Pi
GND → GND

Servo MG995:
VCC (czerwony) → zewnętrzne 5V (NIE z pinu RPi - za duży prąd!)
GND (brązowy) → GND (wspólna masa z RPi)
SIG (pomarańczowy) → GPIO 18 (pin 12, PWM)

Przekaźniki pomp:
Pompa 1 (napełnianie zbiornika) IN → GPIO 24 (pin 18)
Pompa 2 (próbówki)             IN → GPIO 25 (pin 22)
VCC → 5V Raspberry Pi
GND → GND
